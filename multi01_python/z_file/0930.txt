데이터를 확인하기 위해서 numpy 라는 library를 사용했어요.

numpy는 데이터를 n개의 dimension을 가진 행렬로 만들었어요. 이걸 ndarray 라고 해요.
ndarray 중요해요!!
numpy를 기반으로 pandas 라는 library도 사용해봤어요.

pandas는 numpy의 ndarray를 두가지 타입으로 만들었어요.
series 라는 타입은 하나의 dimension을 가진 ndarray + index 래요.
dataframe 이라는 타입은 두개의 dimension을 가진 ndarray + index + column 이래요.

data를 시각화(visualization) 하기 위해서 matplotlib 라는 library도 사용해 봤어요,.
matplotlib 라는 library 는 figure 라는 객체를 가지고, ax를 나눴대요. -=> subplots 라고 해요.

matplotlib를 기반으로 seaborn 이라는 library 도 만들었대요.
dataset도 제공해 주더라구요.

plotly 라는 library도 봤는데, interactive 하게 만들어져 있대요. 얘로는 dashboard도 만들 수 있대요.
